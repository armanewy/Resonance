# Self-Funded OfferLab Experiment Simulator

The simulator compares rough economics for:

- No experiment/current policy.
- Shadow decision support.
- Two-policy randomized test.
- Multi-price randomized test.

Inputs include listings, item cost, asking price, baseline sale probability,
baseline sale price, fees, shipping, holding cost, horizon, setup cost, and max
discount.

Outputs include expected units sold, sale price, margin per sale, holding cost,
setup cost, total expected contribution margin, and incremental margin versus no
experiment.

This is a spending-planning tool, not evidence that an experiment will work. Use
observed store data before buying inventory or changing listing policy.
